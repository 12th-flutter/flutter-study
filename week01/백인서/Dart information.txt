dart
- 객체지향 언어
- 구글에서 제작 됨
- User Interface를 만드는 데에 최적화 되어있음(속도가 빠름 많은 플랫폼에 컴파일 가능)
- flutter를 배우기 위한 언어: 최적화가 되기 좋음

- 두개의 컴파일러
1. dart web: dart code > javascript로 변환 
2. dart Native: dart code > CPU의 아키택쳐에 맞게 변환 : 다양한 플랫폼에 컴파일 가능 ex) iot 포함

- just in time(JIT): dart vm를 활용해 바로 보여줌 / 조금 느릴 수 있음(가상 머신에서 쓰이기 때문에) >> 개발 중 유용
- ahead of time(AOT): 컴파일 후 바이너리 배포 > 시간이 오래걸림, 각 플랫폼에 맞게 해야함 ex) 하나를 수정하려면 처음부터 모든걸 컴파일 해야함

따라서 개발중에는 jit 개발 후에는 aot

null safety: null을 참조하게 하여 더 안전하게 함
